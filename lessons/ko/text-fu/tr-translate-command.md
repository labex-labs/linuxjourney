## Lesson Content

`tr` (translate) 명령어는 한 문자 집합을 다른 문자 집합으로 변환할 수 있게 해줍니다. 모든 소문자를 대문자로 변환하는 예시를 살펴보겠습니다.

```bash
$ tr a-z A-Z
hello
HELLO
```

보시다시피, `a-z` 범위를 `A-Z`로 만들었고, 입력하는 모든 소문자 텍스트가 대문자로 변환됩니다.

hello

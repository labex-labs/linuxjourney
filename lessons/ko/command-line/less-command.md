## Lesson Content

한 화면에 다 들어가지 않는 큰 텍스트 파일을 볼 때 `less` 명령어는 매우 유용한 도구입니다. 오래된 유닉스 격언처럼 "less is more"라는 말이 있는데, 이는 비슷한 기능을 가진 `more` 명령어도 있기 때문입니다.

`less` 유틸리티는 텍스트를 페이지 단위로 보여주어 터미널이 넘치지 않고 파일을 탐색할 수 있게 해줍니다.

### less 명령어 시작하기

파일을 보기 위해서는 `less` 다음에 파일 이름을 입력하세요.

```bash
$ less /home/pete/Documents/text1
```

`less` 뷰어 안에 들어가면 일반 셸 명령어는 작동하지 않습니다. 대신 특정 키들을 사용해 텍스트를 탐색하고 조작합니다.

### 탐색 및 조작법

문서 내에서 이동할 때 다음 키들을 사용할 수 있습니다:

- **화살표 키와 페이지 키**: `Page Up`, `Page Down`, `Up`, `Down` 키로 한 줄씩 또는 한 페이지씩 이동합니다.
- **처음으로 이동**: `g` 키를 눌러 텍스트 파일의 시작 부분으로 바로 이동합니다.
- **끝으로 이동**: `G` (Shift + g) 키를 눌러 텍스트 파일의 끝으로 점프합니다.
- **반 페이지 이동**: `u` 키로 위로, `d` 키로 아래로 반 페이지씩 이동합니다.
- **도움말 메뉴**: `less` 안에서 명령어가 기억나지 않으면 `h` 키를 눌러 도움말 요약을 볼 수 있습니다.

### less에서 검색하기

`less`의 강력한 기능 중 하나는 텍스트 검색입니다. `/` 다음에 찾고 싶은 텍스트를 입력하고 Enter를 누르세요.

- `/search_term`: "search_term"을 앞으로 검색합니다.
- `?search_term`: "search_term"을 뒤로 검색합니다.
- `n`: 다음 검색 결과로 이동합니다.
- `N`: 이전 검색 결과로 이동합니다.

### less 종료 방법

파일 보기 작업을 마쳤으면 `less`를 종료하고 명령 프롬프트로 돌아가는 방법을 알아야 합니다.

- **종료**: `q` 키를 눌러 `less` 뷰어를 종료하고 셸로 돌아갑니다.

### 유용한 less 옵션

옵션을 붙여서 `less`를 시작할 수 있습니다:

```bash
$ less -N file.txt
$ less +G file.txt
$ less +F /var/log/syslog
```

- `-N`: 줄 번호를 표시합니다.
- `+G`: 파일 끝에서 시작합니다.
- `+F`: `tail -f`와 비슷하게 새로 추가되는 내용을 따라갑니다.

`+F` 옵션으로 파일을 따라가다가 `Ctrl-C`를 눌러 따라가기를 멈추고 일반 탐색 모드로 돌아간 후, `q`를 눌러 종료할 수 있습니다.

### 자주 묻는 질문

**less가 cat보다 나은가요?** 짧은 파일은 `cat`을, 긴 파일이나 검색이 필요한 파일은 `less`를 사용하세요.

**대소문자 구분 없이 검색하려면?** `-i` 옵션을 붙여 `less`를 시작하거나, 많은 시스템에서 대문자가 없는 패턴으로 검색하면 자동으로 대소문자 구분 없이 검색됩니다.

**less가 명령어 출력을 열 수 있나요?** 네, 가능합니다. 예를 들어 `dmesg | less`처럼 출력 결과를 파이프로 넘겨서 볼 수 있습니다.

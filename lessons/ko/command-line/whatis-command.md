## Lesson Content

지금까지 꽤 많은 명령어를 배웠습니다! 어떤 명령어가 무엇을 하는지 의심스러울 때 `whatis` 명령어를 사용할 수 있습니다. `whatis` 명령어는 명령줄 프로그램에 대한 간략한 설명을 제공합니다.

```bash
whatis cat
```

설명은 각 명령어의 매뉴얼 페이지에서 가져옵니다. `whatis cat`을 실행하면 짧은 설명이 담긴 작은 문구를 볼 수 있습니다.

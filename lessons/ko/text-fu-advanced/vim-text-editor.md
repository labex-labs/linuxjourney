## Lesson Content

Vim 은 Vi Improved 의 약자입니다. 이름에서 알 수 있듯이 Vi 텍스트 편집기의 개선된 버전입니다.

매우 가볍습니다. Vim 으로 파일을 열고 편집하는 것은 빠르고 쉽습니다. 또한 거의 항상 사용할 수 있습니다. 임의의 Linux 배포판을 부팅하면 Vim 이 기본적으로 설치되어 있을 가능성이 높습니다.

Vim 을 실행하려면 다음을 입력하십시오:

```bash
vim
```

## Lesson Content

### Emacs 종료하기

```
C-x C-c
```

열려 있는 버퍼가 있다면, Emacs 를 종료하기 전에 저장할지 묻습니다.

### 혼란스러우신가요?

```
C-h C-h : help menu
```

### 실행 취소 (Undo)

```
C-x u
```

보시다시피, Emacs 는 더 많은 구성 요소를 가지고 있어 학습 곡선이 약간 더 가파릅니다. 하지만 그 대가로 매우 강력한 텍스트 편집기를 얻을 수 있습니다.

## Lesson Content

`ls` 명령을 사용하여 파일과 디렉토리를 나열하는 것처럼, 장치에 대한 정보를 나열하는 유사한 도구를 사용할 수 있습니다.

### USB 장치 나열

```bash
lsusb
```

### PCI 장치 나열

```bash
lspci
```

### SCSI 장치 나열

```bash
lsscsi
```

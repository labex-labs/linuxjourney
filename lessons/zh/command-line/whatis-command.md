## Lesson Content

哇，到目前为止我们已经学到了不少命令了！如果你对某个命令的功能感到疑惑，可以使用 `whatis` 命令。`whatis` 命令提供命令行程序的简要说明。

```bash
whatis cat
```

描述来源于每个命令的手册页。如果你运行 `whatis cat`，你会看到一个简短的描述。

## Lesson Content

我们几乎完成了文件导航，但首先，让我们学习如何读取文件。一个简单的命令是 `cat` 命令，它是 concatenate 的缩写。它不仅可以显示文件内容，还可以合并多个文件并显示它们的输出。

```bash
cat dogfile birdfile
```

它不适合查看大文件，只适用于短内容。我们将在下一课中讨论许多其他用于查看更大文本文件的工具。

## Exercise

对不同的文件和目录运行 `cat`。然后尝试 `cat` 多个文件。

## Lesson Content

就像我们使用 `ls` 命令列出文件和目录一样，我们可以使用类似的工具来列出设备信息。

### 列出 USB 设备

```bash
lsusb
```

### 列出 PCI 设备

```bash
lspci
```

### 列出 SCSI 设备

```bash
lsscsi
```

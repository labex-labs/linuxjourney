## Lesson Content

要在 Vim 会话中搜索表达式，只需键入`/`键，然后输入您的搜索词。按下 Enter 键后，您可以按`n`向前或按`N`向后浏览搜索结果。

```plaintext
My pretty file is very pretty.

/pretty

will find the pretty words in the text file.
```

`?`搜索命令将向后搜索文本文件。因此，在前面的示例中，最后一个“pretty”将首先出现。

```plaintext
My pretty file is very pretty.

?pretty

will find the pretty words in the text file.
```

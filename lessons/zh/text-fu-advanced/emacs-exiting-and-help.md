## Lesson Content

### To close out of Emacs

```
C-x C-c
```

如果您有任何打开的缓冲区，Emacs 会在关闭前询问您是否保存。

### Confused?

```
C-h C-h : help menu
```

### Undo

```
C-x u
```

如您所见，Emacs 有更多活动部件，因此学习曲线会陡峭一些。然而，作为回报，您将获得一个非常强大的文本编辑器。

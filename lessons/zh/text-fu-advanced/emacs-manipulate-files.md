## Lesson Content

在许多（如果不是所有）Emacs 文档中，您会看到 C-[字母] 语法。这仅仅意味着按下 Ctrl-字母，但为了简写，我们将 Ctrl 称为 C。如果您看到 M-[字母] 这样的语法，则表示使用 Meta 键，最常见的是 Alt 键。

### Saving files

```
C-x C-s - Save a file
C-x C-w - Save file as
C-x s - Save all
```

保存文件选项会提示您是否要保存每个文件。

### Opening a file

```
C-x C-f
```

这将提示您输入要打开的文件名。如果您没有已存在的文件，它将创建一个新文件。您也可以加载一个目录。

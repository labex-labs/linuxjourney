## Lesson Content

Vim 代表 Vi Improved。顾名思义，它是 Vi 文本编辑器的改进版本。

它非常轻量级；使用 Vim 打开和编辑文件既快速又简单。它也几乎总是可用。如果你启动一个随机的 Linux 发行版，Vim 很可能默认安装。

要启动 Vim，只需输入：

```bash
vim
```

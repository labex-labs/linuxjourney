## Lesson Content

`tr` (translate) 命令允许您将一组字符转换为另一组字符。让我们尝试一个将所有小写字符转换为大写字符的示例。

```bash
$ tr a-z A-Z
hello
HELLO
```

如您所见，我们将 `a-z` 的范围转换为 `A-Z`，所有我们输入的小写文本都变成了大写。

## Exercise

尝试以下命令。会发生什么？

```bash
$ tr -d ello
hello
```

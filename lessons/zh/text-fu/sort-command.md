## Lesson Content

`sort` 命令用于对行进行排序。

```plaintext
file1.txt
dog
cow
cat
elephant
bird

$ sort file1.txt
bird
cat
cow
dog
elephant
```

你也可以进行反向排序：

```bash
$ sort -r file1.txt
elephant
dog
cow
cat
bird
```

也可以按数值排序：

```bash
$ sort -n file1.txt
bird
cat
cow
elephant
dog
```

## Exercise

`sort` 命令真正的强大之处在于它能与其他命令结合使用。尝试以下命令，看看会发生什么：

```bash

```

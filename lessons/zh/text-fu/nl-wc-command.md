## Lesson Content

`wc`（word count，字数统计）命令显示文件中单词的总数。

```bash
$ wc /etc/passwd
 96     265    5925 /etc/passwd
```

它分别显示行数、单词数和字节数。

要只查看某个字段的计数，请分别使用 `-l`、`-w` 或 `-c` 选项。

```bash
$ wc -l /etc/passwd
96
```

另一个可以用来检查文件中行数的命令是 `nl`（number lines，行号）命令。

```plaintext
file1.txt
i
like
turtles
```

```bash
$ nl file1.txt
1. i
2. like
3. turtles
```

## Lesson Content

Изучая командную строку Linux, вы столкнетесь с огромным количеством команд. Естественно, можно забыть, что делает конкретная команда. К счастью, существует простая утилита, которая поможет вам.

### Что такое команда whatis

Команда `whatis` выводит краткое, однострочное описание команды непосредственно из её man-страницы. Это быстрый способ напомнить себе основное назначение команды, не читая всю man-страницу.

### Как использовать команду whatis

Использовать `whatis` очень просто. Введите `whatis`, а затем команду, о которой хотите узнать.

```bash
$ whatis cat
cat (1)              - concatenate files and print on the standard output
```

### Понимание вывода

Описание, предоставляемое `whatis`, берётся из раздела `NAME` man-страницы команды. Если у команды есть несколько man-страниц в разных разделах, `whatis` может вывести несколько строк.

```bash
$ whatis passwd
passwd (1)           - change user password
passwd (5)           - the password file
```

Число в скобках — это раздел man-страницы.

### Отличия whatis, man и apropos

- `whatis ls`: Показывает однострочное описание для точного имени команды.
- `man ls`: Открывает полную man-страницу.
- `apropos keyword`: Ищет в описаниях man-страниц по ключевому слову.

Например:

```bash
$ apropos password
```

Используйте `whatis`, когда знаете имя команды, но забыли, что она делает.

### Часто задаваемые вопросы

**Почему whatis выводит "nothing appropriate"?** Возможно, для команды не установлена man-страница или нужно обновить базу данных man.

**Показывает ли whatis опции команды?** Нет. Для опций используйте `man COMMAND` или `COMMAND --help`.

**Является ли whatis тем же, что и which?** Нет. `whatis` описывает команду. `which` показывает путь к исполняемому файлу.

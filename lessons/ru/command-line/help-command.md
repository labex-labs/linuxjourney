## Lesson Content

При работе в командной строке Linux вам часто понадобится быстро освежить в памяти, как работает команда или какие параметры она принимает. Linux предоставляет несколько инструментов помощи прямо в терминале.

### Команда 'help' для встроенных команд Bash

Одним из самых прямых инструментов является `help` — команда, встроенная непосредственно в оболочку Bash. Она специально предназначена для предоставления информации о других встроенных командах Bash. Встроенная команда является частью самой оболочки, а не отдельной программой. Примеры включают `echo`, `cd` и `pwd`.

Чтобы использовать `help`, введите её, а затем имя встроенной команды.

```bash
$ help echo
```

Это отобразит краткое описание команды `echo`, её синтаксис и список доступных опций. Это самый быстрый способ получить помощь по функциям, специфичным для оболочки.

### Флаг --help для исполняемых программ

Для большинства других исполняемых программ, которые не встроены в оболочку, команда `help` не сработает. Вместо этого распространённой практикой является предоставление флага `--help`. Эта опция заставляет программу вывести краткое руководство по использованию и завершить работу.

```bash
$ ls --help
```

Хотя большинство разработчиков придерживаются этого стандарта, он не является универсальным. Попытка использовать `--help` обычно является хорошим первым шагом для незнакомой программы.

### Определение типа команды

Если вы не уверены, является ли команда встроенной в Bash или внешней программой, используйте `type`.

```bash
$ type cd
cd is a shell builtin
$ type ls
ls is /usr/bin/ls
```

Это поможет вам выбрать между `help cd`, `ls --help` или `man ls`.

### Выбор правильного инструмента помощи

- Используйте `help COMMAND` для встроенных команд Bash, таких как `cd`, `echo` и `history`.
- Используйте `COMMAND --help` для быстрого обзора многих внешних команд.
- Используйте `man COMMAND` для подробных руководств.
- Используйте `whatis COMMAND` для однострочного описания.

### Часто задаваемые вопросы

**Почему команда help ls не работает?** `ls` обычно является внешней программой, а не встроенной командой Bash. Попробуйте `ls --help` или `man ls`.

**Почему --help не работает для каждой команды?** Это соглашение, а не строгое правило.

**Какой самый быстрый способ проверить команду?** Попробуйте `COMMAND --help` для внешних команд и `help COMMAND` для встроенных команд Bash.

## Lesson Content

Что ж, вы отлично справились с основами. Мы только прикоснулись к поверхности; теперь, когда вы научились ползать, в следующем наборе курсов я научу вас ходить.

А пока можете похлопать себя по плечу и отдохнуть. Чтобы выйти из оболочки, вы можете использовать команду `exit`:

```bash
exit
```

Или команду `logout`:

```bash
logout
```

Или, если вы работаете в графическом интерфейсе терминала, вы можете просто закрыть терминал. До встречи на следующем курсе!

## Lesson Content

Теперь, когда мы знаем, как перемещаться по файловой системе, как узнать, что у нас доступно? Команда `ls` выводит список файлов и каталогов, чтобы вы могли просмотреть текущее расположение или другой путь.

### Основное использование команды ls

По умолчанию команда `ls` выводит каталоги и файлы в вашем текущем каталоге. Однако вы также можете указать путь, чтобы просмотреть содержимое другого каталога.

```bash
$ ls
$ ls /home/pete
```

Вы также можете вывести конкретный файл:

```bash
$ ls /etc/hosts
/etc/hosts
```

### Просмотр скрытых файлов

Не все файлы в каталоге видны по умолчанию. В Linux имена файлов, начинающиеся с точки (`.`), считаются скрытыми. Вы можете просмотреть их с помощью опции `-a`, которая означает «все».

```bash
$ ls -a
.  ..  .bashrc  Documents  Pictures
```

### Получение подробной информации

Еще одна важная опция `ls` — `-l` для длинного формата. Она показывает права доступа к файлу, количество ссылок, владельца, группу, размер, время изменения и имя.

```bash
$ ls -l
```

Вот пример вывода:

```plaintext
pete@icebox:~$ ls -l
total 80
drwxr-x--- 7 pete penguingroup   4096 Nov 20 16:37 Desktop
drwxr-x--- 2 pete penguingroup   4096 Oct 19 10:46  Documents
drwxr-x--- 4 pete penguingroup   4096 Nov 20 09:30 Downloads
drwxr-x--- 2 pete penguingroup   4096 Oct  7 13:13   Music
drwxr-x--- 2 pete penguingroup   4096 Sep 21 14:02 Pictures
drwxr-x--- 2 pete penguingroup   4096 Jul 27 12:41   Public
drwxr-x--- 2 pete penguingroup   4096 Jul 27 12:41   Templates
drwxr-x--- 2 pete penguingroup   4096 Jul 27 12:41   Videos
```

Для удобного отображения размеров добавьте `-h` для удобочитаемого формата:

```bash
$ ls -lh
```

### Сортировка в обратном порядке

Иногда нужно изменить порядок сортировки. Опция `-r` выводит файлы и каталоги в обратном порядке.

```bash
$ ls -r
```

Вы можете сортировать по времени изменения с помощью `-t`, а затем обратить порядок с помощью `-r`:

```bash
$ ls -lt
$ ls -ltr
```

### Комбинирование флагов команд

Команды имеют флаги, также называемые опциями, для добавления функциональности. Как мы видели с `-a` и `-l`, их можно комбинировать в одной команде, например `ls -la`. Порядок флагов часто не имеет значения, поэтому `ls -al` работает так же.

```bash
$ ls -la
```

Полезные комбинации включают:

```bash
$ ls -lh
$ ls -la
$ ls -ltr
```

### Распространённые опции ls

- `-a`: Показать все файлы, включая скрытые.
- `-l`: Использовать длинный формат.
- `-h`: Показать удобочитаемые размеры с `-l`.
- `-r`: Обратить порядок сортировки.
- `-t`: Сортировать по времени изменения.
- `-S`: Сортировать по размеру файла.
- `-d`: Вывести сам каталог, а не его содержимое.

### Часто задаваемые вопросы

**Почему некоторые имена файлов начинаются с точки?** Файлы с точкой скрыты по умолчанию и часто хранят настройки, например `.bashrc`.

**Как вывести только сам каталог?** Используйте `ls -d directory/`.

**Как увидеть самые новые файлы последними?** Используйте `ls -ltr`.

**Почему ls показывает цвета?** Многие системы настраивают `ls` для цветного отображения типов файлов через алиас или переменную окружения.

## Lesson Content

Ваша оболочка сохраняет запись команд, которые вы вводили ранее. Вы можете получить доступ к этому списку, когда хотите найти и повторно использовать команду без повторного ввода. Команда `history` — это базовый инструмент в Bash и многих Unix-подобных оболочках.

### Просмотр истории команд

Чтобы увидеть список использованных команд, введите `history`.

```bash
$ history
  101  pwd
  102  ls -la
  103  cat notes.txt
```

Каждая строка содержит номер в истории, за которым следует команда.

### Повторный запуск предыдущих команд

Оболочка предоставляет несколько сокращений для упрощения повторного запуска команд.

- **Стрелка вверх**: Хотите выполнить ту же команду, что и только что? Просто нажмите клавишу стрелки вверх, чтобы пройти назад по истории.
- **Сокращение `!!`**: Чтобы выполнить последнюю команду снова, можно использовать `!!`. Например, если вы только что выполнили `cat file1`, ввод `!!` и нажатие Enter запустит `cat file1` снова.
- **Запуск по номеру**: Используйте `!102`, чтобы выполнить команду под номером 102 из истории.
- **Запуск по префиксу**: Используйте `!cat`, чтобы выполнить последнюю команду, начинавшуюся с `cat`.

### Поиск по истории

Одна из самых мощных функций истории — это `Ctrl-R`. Она запускает обратный поиск. После нажатия `Ctrl-R` начните вводить любую часть команды, которую ищете, и оболочка покажет самое последнее совпадение. Вы можете нажимать `Ctrl-R` повторно, чтобы просмотреть более старые совпадения. Когда найдете нужную команду, просто нажмите Enter для её выполнения.

Если хотите отредактировать найденную команду перед запуском, нажмите клавишу стрелки вправо или влево вместо Enter.

### Управление списком истории

Помимо просмотра истории, вы можете управлять ей напрямую.

- **Очистить текущий список истории**: `history -c` удаляет все записи из истории в памяти.
- **Записать историю в файл**: `history -w` сохраняет историю текущей сессии в файл истории, обычно `~/.bash_history`.
- **Удалить конкретную запись**: `history -d <offset>` удаляет команду по её номеру в истории.

Примеры:

```bash
$ history -d 101
$ history -w
```

Будьте осторожны с командами расширения истории, такими как `!!` и `!102`. Сначала используйте `history`, чтобы убедиться, что будет выполнено.

### Другие полезные инструменты терминала

Когда окно терминала заполняется, возможно, захочется очистить его. Используйте команду `clear`, чтобы стереть экран и начать с чистого листа.

```bash
$ clear
```

Ещё одна незаменимая функция — автодополнение по Tab. Если начать вводить начало команды, имени файла или каталога и нажать клавишу Tab, оболочка попытается дополнить ввод. Если вариантов несколько, она может показать их или ничего не сделать. Повторное нажатие Tab часто выведет список всех возможных вариантов.

### Часто задаваемые вопросы

**Где хранится история Bash?** Обычно в `~/.bash_history`, хотя точное поведение зависит от настроек оболочки.

**Включает ли история каждую команду сразу?** Не всегда. Некоторые оболочки записывают историю при выходе из сессии, если не настроено иначе.

**Может ли история содержать конфиденциальные данные?** Да. Избегайте ввода паролей, токенов или секретов напрямую в команды.

**В чём разница между history -c и clear?** `history -c` очищает историю команд в памяти. `clear` очищает только экран терминала.

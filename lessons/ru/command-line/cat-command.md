## Lesson Content

После изучения навигации по файловой системе следующим шагом является просмотр содержимого файлов. Основным и универсальным инструментом для этого является команда `cat`. Название `cat` происходит от слова "concatenate" (объединять), что намекает на её способность связывать файлы вместе.

### Просмотр содержимого файла

Самое простое использование команды `cat` — вывести содержимое одного файла прямо в терминал.

```bash
$ cat myfile.txt
```

Эта команда выведет весь текст из файла `myfile.txt` на экран. Это отлично подходит для коротких конфигурационных файлов или текстовых фрагментов, но не идеально для больших файлов, так как текст будет быстро прокручиваться. Инструменты, лучше подходящие для больших файлов, мы рассмотрим в одном из следующих уроков.

### Объединение файлов

В соответствии с названием, `cat` может объединять несколько файлов и выводить их содержимое последовательно. Она читает файлы в порядке их указания и выводит их один за другим.

```bash
$ cat dogfile birdfile
```

Эта команда сначала выведет содержимое `dogfile`, а затем сразу после него содержимое `birdfile`.

Чтобы сохранить объединённый вывод в новый файл, используйте перенаправление:

```bash
$ cat dogfile birdfile > animals
```

### Создание файлов с помощью перенаправления

Вы также можете использовать `cat` с оператором перенаправления вывода (`>`) для создания новых файлов. Это быстрый способ записать текст в файл прямо из терминала.

```bash
$ cat > newfile.txt
```

После выполнения этой команды вы можете вводить текст. Нажмите `Ctrl+D` на новой строке, чтобы сохранить и выйти. Это создаст файл `newfile.txt` с введённым вами текстом. Будьте осторожны: использование `>` для существующего файла полностью перезапишет его.

Чтобы добавить текст в конец файла, не перезаписывая его, используйте `>>`.

```bash
$ cat >> notes.txt
```

### Распространённые опции команды cat

Команда `cat` имеет несколько опций для изменения своего поведения.

- `-n`: Нумеровать все строки вывода, начиная с 1.
- `-b`: Нумеровать только непустые строки вывода.
- `-s`: Сжимать несколько пустых строк в одну.
- `-A`: Показывать непечатаемые символы, табуляции и окончания строк.

Примеры:

```bash
$ cat -n script.sh
$ cat -b notes.txt
$ cat -s messy.txt
```

### Когда не стоит использовать cat

Используйте `cat` для коротких файлов. Для длинных файлов лучше использовать `less`, чтобы можно было прокручивать, искать и выходить, не засоряя терминал.

```bash
$ less /var/log/syslog
```

### Часто задаваемые вопросы

**Что означает cat?** Это сокращение от concatenate (объединять).

**Можно ли редактировать файл с помощью cat?** Нет, не интерактивно. Она может создавать или перезаписывать файлы с помощью перенаправления, но для редактирования лучше использовать текстовый редактор.

**В чём разница между > и >>?** `>` перезаписывает файл. `>>` добавляет в конец файла.

## Lesson Content

Команда `touch` — это стандартная утилита в операционных системах, похожих на Unix. Хотя её основная задача — изменение временных меток файлов, её также часто используют для создания новых пустых файлов.

Базовый синтаксис:

```bash
touch [OPTIONS] FILE...
```

### Создание новых файлов

Самый простой способ создать пустой файл — использовать `touch`, за которым следует имя файла. Если файл не существует, `touch` создаст его.

```bash
$ touch mysuperduperfile
```

После выполнения этой команды в текущем каталоге появится новый пустой файл с именем `mysuperduperfile`. Вы можете создать несколько файлов одновременно, перечислив их имена.

```bash
$ touch file1.txt file2.txt file3.log
```

Это полезно при настройке структуры проекта или создании файлов-заполнителей перед добавлением содержимого.

### Обновление временных меток файлов

Изначальная функция `touch` — обновлять временные метки доступа и изменения файла или каталога. Если использовать `touch` для существующего файла, он обновит его временные метки на текущее время.

Вы можете проверить это с помощью `ls -l`, чтобы увидеть временную метку файла, затем выполнить `touch` и проверить снова.

```bash
# Check the original timestamp
$ ls -l mysuperduperfile

# Update the timestamp
$ touch mysuperduperfile

# Check the new timestamp
$ ls -l mysuperduperfile
```

### Расширенное управление временными метками

Команда `touch` также предоставляет опции для более точного управления временными метками.

Используйте файл-образец с опцией `-r`, чтобы скопировать временные метки с одного файла на другой.

```bash
$ touch -r file1.txt file2.txt
```

Установите конкретную дату и время с помощью `-d`:

```bash
$ touch -d "2026-06-23 12:30:00" mysuperduperfile
```

Используйте `-c`, если хотите обновить файл только в случае его существования. С `-c` `touch` не создаст отсутствующий файл.

```bash
$ touch -c existing-file.txt
```

### Распространённые опции touch

- `-a`: изменить только время доступа.
- `-m`: изменить только время изменения.
- `-c`: не создавать файл, если он не существует.
- `-d "DATE"`: использовать конкретную строку даты.
- `-r FILE`: использовать временные метки другого файла в качестве образца.
- `-t STAMP`: использовать временную метку в компактном числовом формате.

Например, эта команда изменит только время изменения:

```bash
$ touch -m notes.txt
```

### Часто задаваемые вопросы

**Добавляет ли touch текст в файл?** Нет. `touch` создаёт пустой файл или обновляет временные метки. Для добавления текста используйте редактор, `echo` или `cat`.

**Перезапишет ли touch существующий файл?** Нет. Он обновляет временные метки, но не заменяет содержимое файла.

**Зачем использовать touch в скриптах?** Это быстрый способ убедиться, что файл существует, или отметить, что задача была выполнена в определённое время.

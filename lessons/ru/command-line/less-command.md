## Lesson Content

При просмотре текстовых файлов, которые слишком велики, чтобы уместиться на одном экране, команда `less` становится незаменимым инструментом. Как гласит старая пословица Unix: «less is more» (меньше — значит больше). Это игра слов, поскольку существует также команда `more` с похожим функционалом.

Утилита `less` отображает текст в постраничном формате, позволяя вам перемещаться по файлу, не засоряя терминал.

### Начало работы с командой Less

Чтобы начать просмотр файла, используйте `less`, за которым следует имя файла.

```bash
$ less /home/pete/Documents/text1
```

Когда вы находитесь внутри просмотрщика `less`, стандартные команды оболочки не работают. Вместо этого используется специальный набор клавиш для навигации и взаимодействия с текстом.

### Навигация и управление

Вы можете использовать несколько клавиш для перемещения по документу:

- **Стрелки и клавиши страницы**: Используйте `Page Up`, `Page Down`, `Up` и `Down` для перемещения построчно или постранично.
- **Перейти к началу**: Нажмите `g`, чтобы перейти непосредственно к началу текстового файла.
- **Перейти к концу**: Нажмите `G` (Shift + g), чтобы перейти к концу текстового файла.
- **Перемещение на полстраницы**: Нажмите `u`, чтобы подняться вверх, и `d`, чтобы опуститься вниз.
- **Меню помощи**: Если вы забыли команды внутри `less`, просто нажмите `h` для отображения полезной сводки.

### Поиск в Less

Мощная функция `less` — возможность поиска текста. Введите `/`, за которым следует искомый текст, затем нажмите Enter.

- `/search_term`: Поиск вперед по "search_term".
- `?search_term`: Поиск назад по "search_term".
- `n`: Переход к следующему вхождению поискового термина.
- `N`: Переход к предыдущему вхождению.

### Как выйти из Less

Когда вы закончите просмотр файла, нужно знать, как `выйти из less` и вернуться к командной строке.

- **Выйти**: Просто нажмите `q`, чтобы выйти из просмотрщика `less` и вернуться в оболочку.

### Полезные опции less

Вы можете запускать `less` с опциями:

```bash
$ less -N file.txt
$ less +G file.txt
$ less +F /var/log/syslog
```

- `-N`: Показывать номера строк.
- `+G`: Открыть файл с конца.
- `+F`: Следить за добавлением нового содержимого, аналогично `tail -f`.

При отслеживании файла с помощью `+F` нажмите `Ctrl-C`, чтобы прекратить слежение и вернуться к обычной навигации, затем нажмите `q` для выхода.

### Часто задаваемые вопросы

**less лучше, чем cat?** Используйте `cat` для коротких файлов и `less` для длинных файлов или тех, в которых нужно искать.

**Как искать без учета регистра?** Запускайте `less` с опцией `-i` или вводите поисковые запросы без заглавных букв — на многих системах это обеспечивает нечувствительный к регистру поиск.

**Можно ли открыть вывод команды через less?** Да. Передайте вывод через конвейер, например `dmesg | less`.

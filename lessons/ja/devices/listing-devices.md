## Lesson Content

`ls`コマンドを使ってファイルやディレクトリを一覧表示するのと同じように、デバイスに関する情報を一覧表示する同様のツールを使用できます。

### Listing USB Devices

```bash
lsusb
```

### Listing PCI Devices

```bash
lspci
```

### Listing SCSI Devices

```bash
lsscsi
```

## Exercise

これらの各コマンドを試して、どのような出力が得られるかを確認してください。

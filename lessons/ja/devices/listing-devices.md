## Lesson Content

`ls`コマンドを使ってファイルやディレクトリを一覧表示するのと同じように、デバイスに関する情報を一覧表示する同様のツールを使用できます。

### USB デバイスのリスト表示

```bash
lsusb
```

### PCI デバイスのリスト表示

```bash
lspci
```

### SCSI デバイスのリスト表示

```bash
lsscsi
```

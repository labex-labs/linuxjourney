## Lesson Content

Vim は Vi Improved の略です。その名の通り、Vi テキストエディタの改良版です。

非常に軽量で、Vim でファイルを開いたり編集したりするのは素早く簡単です。また、ほとんどの場合利用可能です。適当な Linux ディストリビューションを起動すると、Vim がデフォルトでインストールされている可能性が高いです。

Vim を起動するには、次のように入力します。

```bash
vim
```

## Lesson Content

### Emacs を終了するには

```
C-x C-c
```

開いているバッファがある場合、Emacs を閉じる前に保存するかどうか尋ねられます。

### 困ったときは？

```
C-h C-h : help menu
```

### 元に戻す

```
C-x u
```

ご覧の通り、Emacs にはより多くの可動部分があるため、学習曲線は少し急です。しかしその代わりに、非常に強力なテキストエディタを手に入れることができます。

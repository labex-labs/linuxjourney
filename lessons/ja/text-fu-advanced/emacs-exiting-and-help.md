## Lesson Content

### Emacs を終了するには

```
C-x C-c
```

開いているバッファがある場合、Emacs を終了する前に保存するかどうかを尋ねられます。

### 困ったときは？

```
C-h C-h : help menu
```

### 元に戻す

```
C-x u
```

ご覧のとおり、Emacs にはより多くの機能があるため、学習曲線は少し急です。しかし、その代わりに非常に強力なテキストエディタを手に入れることができます。

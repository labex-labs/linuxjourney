## Lesson Content

Linuxでは、ほぼすべてのコマンドに専用の説明書が付属しています。これを「manページ」（manual pagesの略）と呼び、システムを効果的に使うための重要なリソースです。

### manページの理解

manページはLinuxのコマンド、ユーティリティ、システムコールの組み込みドキュメントです。コマンドの動作内容、利用可能なオプション（フラグ）、使い方を詳しく説明しています。コマンドラインのヘルプを得るための最初で最良の情報源です。

### manでマニュアルを表示する

任意のコマンドのマニュアルを見るには、`man`に続けてコマンド名を入力します。例えば、`ls`のマニュアルを読むには次のようにします。

```bash
$ man ls
```

これで`ls`のmanページが開きます。矢印キーでスクロールし、`/`で検索、`q`で終了できます。

### コマンドオプションの詳細を調べる

manページはコマンドのオプションを理解するのに特に役立ちます。例えば、`ls -l`を見かけて`-l`の意味を知りたい場合は、`man ls`を開いて`-l`を検索してください。

manページ内では：

- `/`を押して検索語を入力すると前方検索ができます。
- `n`で次の一致箇所へジャンプ。
- `N`で前の一致箇所へジャンプ。
- `q`で終了。

### manページのセクションの理解

マニュアルページは番号付きのセクションに分かれています。よく使われるセクションは：

- `1`: ユーザーコマンド
- `2`: システムコール
- `3`: ライブラリ関数
- `5`: ファイルフォーマット
- `8`: システム管理コマンド

同じ名前のコマンドが複数のセクションに存在することがあります。その場合はセクション番号を指定できます。

```bash
$ man 5 passwd
$ man 1 passwd
```

### よくある質問

**なぜmanの出力は長いのですか？** manページは参照用ドキュメントです。必要な部分にジャンプするにはman内の検索を使いましょう。

**manを終了するには？** `q`を押します。

**manページが存在しない場合は？** `COMMAND --help`、`help COMMAND`を試すか、ディストリビューションのドキュメントパッケージをインストールしてください。

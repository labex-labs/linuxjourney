## Lesson Content

1画面に収まらない大きなテキストファイルを閲覧する際、`less` コマンドは非常に便利なツールです。古いUnixのことわざに「less is more（lessはmoreより良い）」という言葉があります。これは、似た機能を持つ `more` コマンドも存在することにかけた言葉遊びです。

`less` ユーティリティはテキストをページ単位で表示し、端末を大量のテキストで埋め尽くすことなくファイル内を移動できます。

### lessコマンドの使い始め

ファイルを閲覧するには、`less` に続けてファイル名を指定します。

```bash
$ less /home/pete/Documents/text1
```

`less` ビューア内に入ると、通常のシェルコマンドは使えません。代わりに特定のキー操作でテキストを操作します。

### ナビゲーションと操作方法

ドキュメント内を移動するために使えるキーは以下の通りです：

- **矢印キーとページキー**: `Page Up`、`Page Down`、`Up`、`Down` を使って行単位またはページ単位で移動します。
- **先頭へ移動**: `g` を押すとファイルの先頭に直接移動します。
- **末尾へ移動**: `G`（Shift + g）を押すとファイルの末尾にジャンプします。
- **半ページ移動**: `u` で上へ、`d` で下へ半ページ移動します。
- **ヘルプメニュー**: `less` 内でコマンドを忘れたら、`h` を押すとヘルプの概要が表示されます。

### lessでの検索

`less` の強力な機能の一つにテキスト検索があります。`/` に続けて検索したい文字列を入力し、Enterを押します。

- `/search_term`: 「search_term」を前方に検索します。
- `?search_term`: 「search_term」を後方に検索します。
- `n`: 次の検索結果にジャンプします。
- `N`: 前の検索結果にジャンプします。

### lessの終了方法

ファイルの閲覧が終わったら、`less` を終了してコマンドプロンプトに戻る方法を知っておく必要があります。

- **終了**: `q` を押すだけで `less` ビューアを終了し、シェルに戻れます。

### 便利なlessオプション

`less` はオプションを付けて起動できます：

```bash
$ less -N file.txt
$ less +G file.txt
$ less +F /var/log/syslog
```

- `-N`: 行番号を表示します。
- `+G`: ファイルの末尾から開きます。
- `+F`: 新しい内容が追加されるのを追跡し、`tail -f` のように動作します。

`+F` で追跡中に `Ctrl-C` を押すと追跡を停止して通常のナビゲーションに戻り、その後 `q` で終了できます。

### よくある質問

**lessはcatより優れているの？** 短いファイルには `cat` を使い、長いファイルや検索が必要な場合は `less` を使いましょう。

**大文字小文字を区別せずに検索するには？** `-i` オプションを付けて `less` を起動するか、多くのシステムでは大文字を含まないパターンで検索すると自動的に無視されます。

**lessはコマンドの出力も開ける？** はい。例えば `dmesg | less` のようにパイプで渡せます。

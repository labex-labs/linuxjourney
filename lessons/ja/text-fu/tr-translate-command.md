## Lesson Content

`tr` (translate) コマンドは、ある文字セットを別の文字セットに変換することができます。すべての小文字を大文字に変換する例を試してみましょう。

```bash
$ tr a-z A-Z
hello
HELLO
```

ご覧のとおり、`a-z` の範囲を `A-Z` にし、入力したすべての小文字が大文字になります。
